Frequently Asked Questions

Essential requirements to run the program?

OS Supported: Windows 10/11

Installation required:

  • python 3.7 or more
  • lastest pip

Install the following lib with pip on cmd (recommended version):

  • cmake :

    pip install cmake==3.22.3

  • dlib ver.19.22 (modified, special thanks to jloh02 on github) :

    pip install modules/dlib/dlib-19.22.99-cp310-cp310-win_amd64.whl

  • TensorFlow :

    pip install tensorflow==2.8.0

  • OpenCV-Python :

    pip install opencv-python==4.5.5.64

  • Face-Recognition :

    pip install face-recognition==1.3.0

  • MTCNN :

    pip install mtcnn==0.1.1

  • NumPy version 1.22.0 :

    pip install numpy==1.22.0

You must install all to run this.

Do's and Don'ts

Do's

  • One person per run
  • Open 'Attendance.csv' file on Visual Studio Code for real time update
  • For the first usage (new face face), register the face first before running attendance function

Don't

  • Do not open 'Attendance.csv' and/or 'Attendance.xlsx' file on spreadsheet program while the program is running
How to register face?

There are two ways to register your face:

By using the application

  • Open destination folder (default path: img/samples)
  • Open the main program (.exe)
  • click 'face registration' button
  • enter the face name (one person per runs)
  • click register photos
  • If all set, press 'enter' on your keyboard and wait the program to write the photos into the specified path
  • Check if the photo is registered in the folder
  • Mandatory: Click Main Menu to go back (this will automatically reset the name variable)

By manual input

  • Open destination folder (default storage: img/samples)
  • Copy and Paste the image (mandatory: one photo per person, preferably: clean photo, min: 640x480px)

How to use attendance program?

!important: you need to register at least one face before you run this function.

  • Open the main program (.exe)
  • Click 'attendance' button
  • Remember the rules: one person per runs
  • If all set, wait until the program recognizes your face
  • After the program capture your face, press 'esc' again
  • Check the attendance.csv
    • note: Never open 'Attendance.csv' and 'Attendance.xlsx' file on spreadsheet program (ex: Ms. Excel) when the program running.
